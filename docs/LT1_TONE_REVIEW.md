## Стоп-лист на ревью (`tone.stoplist.json` → `VK_ADDENDUM`)

🔴 **Несущий гейт.** Этот список гейтит lore (и далее всю прозу Хранителя). Слишком
широкий/неверный терм **заблокирует легитимную прозу** — нужен глаз Ивана на каждый.
**Принцип severity (§2.3 целит во фразы-штампы, не в одиночные слова):** однозначный
пастиш-фраза + явные канон-маркеры → `block`; неоднозначное нормальное одиночное
слово → `warn` или вон. Поле `severity` в `tone.stoplist.json` — подготовленный
дефолт; финал по каждому терму проставляешь ты.

- [ ] `древнее зло пробуждается` — block (фраза)
- [ ] `тьма сгущается` — block (фраза)
- [ ] `тьма грядёт` — block (фраза)
- [ ] `силы тьмы` — block (фраза)
- [ ] `судьба мира` — block (фраза)
- [ ] `эпическая битва` — block (фраза)
- [ ] `пророчество гласит` — block (фраза)
- [ ] `воистину` — block (одиночное, но явный канон-маркер mock-архаики)
- [ ] `избранный` — **warn** (неоднозначное одиночное слово; реши: warn или вон)
- [ ] `века и века` — warn
