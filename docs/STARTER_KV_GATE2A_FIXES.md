- **Анти-паттерны**: ❌ правило по памяти без сверки; ❌ контент в engine/ «временно»; ❌ проза мимо стоп-листов; ❌ правка без подтверждённого скелета; ❌ молчком менять арх-решения; ❌ публикация/монетизация.
