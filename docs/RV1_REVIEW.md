## Что подтверждено
- **Pack-sourcing:** каждый параметр трассируется к `source_text`; выдуманных чисел нет.
  Числовые величины («1к», число залпов) корректно вынесены в дескрипторы/делегированы,
  а не захардкожены (`pre_melee_volleys: "scene_determined"` → `combat.nachalo_boya`).
- **Терминология:** `pandora_box`/`ru`; калек чужих систем нет.
- **Цитирование:** `book`/`section`/`pages` полны во всех 4.
- **Кросс-ссылки:** `related` разрешаются без висячих; §3.7-тройка взаимно согласована
  (−1к врагам ближнего боя, задача «Продвинуться» — двусторонняя ссылка, бонус «+1к за
  знак»).
- **`shadow_path: "wandering"`** ↔ карта-владелец `shadow.ispolzovanie_izyanov`
  (`shadow_paths.wandering` = «Одержимость скитаниями») — реконсиляция ключа
  `wandering_madness → wandering` корректна, не висячая.
