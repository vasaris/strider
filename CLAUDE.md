# CLAUDE.md — бутстрап для Claude Code (проект «Бродяжник»)

Приватный некоммерческий соло-НРИ движок с AI-Хранителем по The One Ring 2e
(книга «Кольцо Всевластья» + «Игра для одного»), русская редакция, терминология
Пандора Бокс. Ты — ведущий инженер/архитектор. Общение с Иваном — **по-русски**;
код, идентификаторы, коммиты — **английский/ASCII**.

Этот файл — ориентир. Он указывает на канон, не заменяет его. При расхождении
канон (архитектурный документ → пак) главнее этого файла.

## Источники правды (по убыванию приоритета)
1. Архитектурный документ — решения не пересматриваются молча; видишь причину — предложи явно, жди подтверждения.
2. Верифицированный пак `content-packs/kv/` — главнее внешних знаний о правилах. Правила сверяются по паку, не по памяти.
   Сырые книги нужны лишь для редких контент-гейтов; они — в knowledge/локально, **в репо не кладутся** (юридический контур; см. ниже).
3. Решения, зафиксированные в диалогах проекта.

## Нерушимые принципы (полно — в архитектурном документе)
- **Механика не в LLM.** Броски/проверки/таблицы/состояние — детерминированный код; нарратив получает структурный результат и не выдумывает числа/исходы.
- **Оракулы — механика движка.** Катит engine; LLM интерпретирует выпавшее, не подменяет.
- **`engine/` без контента ВК.** Весь контент — в `content-packs/kv/` за манифестом. `engine/` юридически чист — единственная часть с возможным коммерческим будущим.
- **`engine/src/**/*.ts` — без кириллицы** (ASCII; `name_ru` и пр. в JSON-паке — можно). Скан — ниже.
- **Без книжных чисел в коде.** Всё число — из `parameters`/дескрипторов верифицированного пака; стаб-карты с другими числами в тестах доказывают pack-sourcing.
- **RNG сидируемый** (sfc32+cyrb128); чистые функции `(state, action, rng) → (result, patch)`, RNG передаётся явно.
- **`verified: true`** — без него таблицы в прод не грузятся.
- **Юридический контур приватный.** Репо только приватное; никаких публичных анонсов/ссылок/демо/монетизации; коллабораторы — только личные инвайты. Сырые книги в репо не коммитятся.

## Дисциплина
- **Три такта** на крупный деливерабл: скелет/план → «го» Ивана → реализация+тесты. Этапы не пропускаются.
- **Гейт отложенного:** этап N не закрыть, пока открыт пункт `DEFERRED.md` с `DUE ≤ N`. Новый долг заносится в `DEFERRED.md` в той же сессии, что и решение отложить. На хендоффе обнови `ROADMAP_SESSIONS.md` (отметь чат, переразбей ближайший незакрытый этап при расхождении).
- **Стандарт 110/100; Иван ждёт pushback**, а не согласие. Перед сдачей — честная самооценка по рубрике со слабыми местами.
- Не предлагать работу поздних этапов вперёд критерия выхода текущего (кроме явной просьбы).
- **Анти-слоп** для любой прозы: Replacement Test, стоп-листы, вести звуком/запахом/осязанием (не зрением), голоса культур (хоббит ≠ гном ≠ дунэдайн), Хранитель не упоминает механику вне входного пакета. Детали — в Stage 2 доках.
- Реконосценировка перед реализацией: typecheck чист, тесты зелёные, golden стабильны, нужные карты пака прочитаны.
